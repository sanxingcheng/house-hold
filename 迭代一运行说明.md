# 迭代一运行说明

## 环境要求

- **JDK 21**
- **Node 18+**（前端）
- **MySQL 8**（创建库 `household` 并执行建表 SQL）
- **Maven 3.9+**
- （可选）**Nacos**：若使用 Nacos 服务发现，需先启动 Nacos；不使用可参考下方「无 Nacos 运行」方式

## 1. 数据库

**只需手动创建数据库一次**（表会在 auth-user 服务启动时自动创建）：

```sql
CREATE DATABASE IF NOT EXISTS household DEFAULT CHARACTER SET utf8mb4;
```

修改 `house-hold-auth-user/src/main/resources/application.yml` 中的 `spring.datasource.url`、`username`、`password` 为你的 MySQL 配置。

auth-user 启动时会自动执行 `schema.sql`（`CREATE TABLE IF NOT EXISTS`），无需手动建表。

## 2. 无 Nacos 运行（开发）

1. **先启动 auth-user 服务**（端口 8081）  
   在项目根目录 `house-hold` 下执行：
   ```bash
   mvn -pl house-hold-auth-user spring-boot:run
   ```

2. **再启动 Gateway**（端口 8080），使用 dev 配置直连 auth-user：
   ```bash
   mvn -pl house-hold-gateway spring-boot:run -Dspring-boot.run.profiles=dev
   ```

3. **启动前端**
   ```bash
   cd house-hold-web
   npm install
   npm run dev
   ```
   访问 http://localhost:5173 ，可进行注册、登录与跳转至概览页。

## 3. 使用 Nacos 运行

1. 启动 Nacos（默认 8848）。
2. 启动 auth-user：`mvn -pl house-hold-auth-user spring-boot:run`
3. 启动 Gateway：`mvn -pl house-hold-gateway spring-boot:run`（无需 dev 配置）
4. 启动前端：`cd house-hold-web && npm install && npm run dev`

## 4. 接口说明

- 注册：`POST http://localhost:8080/auth/register`  
  Body: `{ "username", "password", "birthday", "email?", "phone?" }`
- 登录：`POST http://localhost:8080/auth/login`  
  Body: `{ "username", "password" }`  
  返回：`{ "token", "user": { "id", "username", "birthday" } }`

前端开发时通过 Vite 代理将 `/api` 转发到 `http://localhost:8080`，因此实际请求为 `/api/auth/login` 等。
